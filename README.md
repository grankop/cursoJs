# cursoJs
TP - Seguimiento de paquetes
La aplicacion consiste en el registro y seguimiento de paquetes, con una pantalla para el correo y una para el cliente. En la pantalla del correo se ingresa la guía del paquete, si ya existe se actualiza y si no se crea un nuevo registro, completando en ambos casos de un combo la Sucursal y el estado del envío, se valida que los dos campos tengan valor. Una vez guardado se listan las últimas guias. El guardado es con localStorage.
En la pantalla del cliente se ingresa un numero de guía, se valida que no esté vacío el campo y luego que el numero ingresado exista, en dicho caso se muestran los movimientos de la misma.
